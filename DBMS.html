Database Management System

Data: is a raw collection of characters, integers, etc. that holds some importance when interpreted.

Information: is the processed and meaningful version of data.

Metadata: is the data of data, which describes the tracking info for data.

Data dictionaries: same as metadata.

Database: is a server or location for storing data in sequential order

Table: A listed arrangement of data for organized use is a table

Tuple(s): is an ordered sequence of values. The values can be repeated, but their number is always finite.

Field(s): is an attribute of a record in a database table.

Domain(s): a set of values that can be stored in a column of a database table.

Constraint(s): are rules or conditions that are applied to the data within a database to ensure data integrity, consistency, and adherence to business rules

Key(s): an attribute/a set of attributes that help us identify a row (or tuple) uniquely in a table (or relation).
